# Store screenshots

Five screenshots per language for App Store + Google Play submission.
Sizes: 1290×2796 (iPhone Pro Max) — Google Play accepts the same
asset rescaled.

Numbering convention (matches `client_creative_assets_spec_es.md`):

| #  | Screen                          | Caption (ES)                         | Caption (EN)                          |
|----|---------------------------------|--------------------------------------|---------------------------------------|
| 01 | Couples feed + pineapple top    | Descubre parejas verificadas         | Discover verified couples             |
| 02 | Filters panel                   | Filtros que van con tu vibra         | Filters that match your vibe          |
| 03 | Travel Match                    | Coinciden en el viaje                | Meet on the way                       |
| 04 | Chat / message request          | Conversaciones que importan          | Conversations that matter             |
| 05 | Profile / security              | Privacidad de base                   | Privacy built in                      |

The captions live in the screenshots themselves (overlaid on the
mockup), not in the binary. Whoever does the layout in Figma owns
the typography choice for the overlays.

These files do not get bundled into the app — they only exist here
for tracking + uploading to the stores at submission time.
